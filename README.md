# Product Registration System

Product Registration System is a Spring Boot MVC web application developed to manage product details efficiently.
It allows users to register, view, update, and delete product information using CRUD operations.

## Features

* Product registration form
* Save product details in database
* Display all registered products
* Edit product information
* Delete product records
* Form validation using annotations
* User-friendly interface with Bootstrap

## Technologies Used

* Java
* Spring Boot
* Spring MVC
* Thymeleaf
* Hibernate / JPA
* MySQL
* Maven
* Bootstrap

## Architecture Used

This project follows **MVC Architecture**:

* **Model** → Product entity class
* **View** → Thymeleaf HTML pages
* **Controller** → Handles user requests

## Project Objective

The main objective of this project is to understand Spring Boot MVC flow, database connectivity, and CRUD operations in a real-time product management system.
